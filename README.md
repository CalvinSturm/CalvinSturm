I build local-first agent runtimes, trust-gated tooling, observability dashboards, and performance-oriented desktop software. My work focuses on reliability, reproducibility, operator control, and practical systems design.

## System philosophy

I prefer systems that are:

- local-first where possible
- explicit about permissions and side effects
- observable when things go wrong
- reproducible across runs
- practical instead of over-engineered
- designed to keep operators in control

A lot of modern AI tooling still hides too much runtime behavior behind black boxes and gives operators weak feedback during failure and recovery. I am interested in making those systems more inspectable, more deterministic, and easier to debug.
